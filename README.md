# React Add to Cart (Using Modal)

A simple ReactJS application that fetches products from the Fake Store API and allows users to add items to the cart using a modal interface.

## Features
- Fetch products using Fake Store API
- Display products with image, title, and price
- Add to Cart button
- Prevent duplicate cart entries (shows alert)
- Navbar with live cart count
- Modal to display cart items
- Remove items from cart
- Responsive UI (TailwindCSS)
- Clean component-based structure

## Tech Stack
- React JS
- Tailwind CSS
- JavaScript
- Fake Store API (https://fakestoreapi.com)

## Installation

1. Clone or download the project
2. Install dependencies:
npm install
3. Run the development server:
npm run dev


## API Used
`https://fakestoreapi.com/products`
